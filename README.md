This application helps manage customers, utility services, and billing operations through a simple menu-driven interface while demonstrating core Object-Oriented Programming (OOP) and Data Structures concepts.

# 📌 Project Overview

The Utility & Billing System is designed to simplify the management of utility services such as electricity, water, and gas. It provides secure user authentication, customer management, bill calculation, and record storage in a menu-driven console application.

The project focuses on implementing real-world programming concepts using C++ while applying Data Structures and Object-Oriented Programming principles.

# ⚙️ Installation

## Clone the Repository

```bash
git clone https://github.com/Github23020/Utility_And_Billing_System.git
```

Enter the project directory:

```bash
cd Utility_And_Billing_System
```

---

## Install Required Packages (Ubuntu / WSL)

```bash
sudo apt update

sudo apt install -y \
build-essential \
make \
g++ \
cmake \
git \
nlohmann-json3-dev
```

---

# ▶️ Build and Run

## Build

```bash
make
```

If your Makefile is configured correctly, the executable will be generated automatically.

---

## Run

```bash
./Utility_And_Billing_System
```

If your executable has a different name, replace it with the name generated by your Makefile.
